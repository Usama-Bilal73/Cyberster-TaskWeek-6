# 🔐 Security Log Analyzer & Automation Lab

## 📌 Project Overview
This project demonstrates the use of automation in cybersecurity by implementing both Bash and Python scripts. It focuses on automating network scanning and log analysis to detect potential threats efficiently.

## ⚙️ Features
- 🔍 Automated network scanning using Bash script
- 📊 Log analysis using Python
- 🚨 Detection of SQL Injection (SQLi) attacks
- 🌐 Filtering of local subnet traffic (10.0.2.x)
- 📌 Identification of unique IP addresses
- 🧠 Regex-based attack pattern matching

## 🛠️ Technologies Used
- Python 3
- Bash Scripting
- Kali Linux
- Libraries:
  - `re` (Regex)
  - `Scapy`
  - `Requests`
